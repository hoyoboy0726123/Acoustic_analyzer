# 測試用音檔目錄

此目錄用於存放測試用的音檔樣本。

## 建議的測試音檔

1. **正常風扇噪音** - `fan_normal.wav`
   - 低頻為主的風扇運轉聲

2. **電感嘯叫樣本** - `coil_whine.wav`
   - 6k-12kHz 有明顯高頻成分

3. **高頻共振樣本** - `resonance.wav`
   - 4k-8kHz 有共振峰值

4. **靜音/背景噪音** - `silence.wav`
   - 用於測試基準

## 注意事項

- 音檔格式: WAV
- 取樣率: 44100 或 48000 Hz
- 位元深度: 16-bit 或 24-bit
- 聲道: Mono
- 長度: 10-30 秒
